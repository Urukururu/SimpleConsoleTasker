# SimpleConsoleTasker 📋

Легковаговий та мінімалістичний консольний планувальник завдань, розроблений на C++ з використанням локальної бази даних SQLite. 
Проєкт створено з акцентом на мінімальне споживання системних ресурсів, миттєвий відгук інтерфейсу та повну автономність роботи.

## 🚀 Основний функціонал
* **Створення:** додавання нових завдань із назвою та детальним описом.
* **Читання:** перегляд списку завдань із кольоровою індикацією статусів.
* **Оновлення:** редагування існуючих завдань та зміна їх статусів (Не виконано / В процесі / Виконано).
* **Видалення:** точкове видалення завдань за ID.
* **Захист вводу:** надійна обробка помилок.
* **Легкий імпорт:** перемістіть файл `tasks.db` в одну директерію зі застосунком.

## 🛠 Стек технологій
* **Мова програмування:** C++ 
* **База даних:** SQLite 3
* **Середовище розробки:** Microsoft Visual Studio 2022
* **Інтерфейс:** CLI

## 📦 Встановлення та запуск
Програма є портативною і не потребує встановлення додаткових бібліотек чи залежностей.
1. Перейдіть у розділ [Releases] справа на цій сторінці.
2. Завантажте останню версію архіву `SimpleConsoleTasker.7z`.
3. Розархівуйте архів та запустіть потрібний вам файл.
4. Запустіть файл (база даних `tasks.db` створиться автоматично в одній директорії).
